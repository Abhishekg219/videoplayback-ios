## Features

- [x] Scrub Video 
- [x] Handle play or stop video in main thread
- [x] Play in UITableView 
- [x] Autoplay video    
- [x] HTTPS support
- [x] Live stream support 
- [x] Written in Swift 
- [x] Landscape support
- [x] Cocoapod support 

## Requirements

- iOS 10.0 or later
- Xcode 8.3 or later
